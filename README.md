### keras-emnist-digits
EMNIST Digits classification with Keras

## Installation
1. install [EMNIST Digits](https://www.nist.gov/itl/products-and-services/emnist-dataset)
2. ```pip install -r requirements.txt```

## Usage
```
python train.py
```

## Dependencies
- Python 3.6.x
- Keras 2.3.1
- Tensorflow 2.1.0
- efficientnet
